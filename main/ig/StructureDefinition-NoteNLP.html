Logical Model: Note NLP OMOP Table

Official URL: https://interop.esante.gouv.fr/ig/fhir/eds/StructureDefinition/NoteNLP Version: 0.1.0
Draft as of 2024-07-29 Computable Name: NoteNLP

The NOTE_NLP table encodes all output of NLP on clinical notes. Each row represents a single extracted term from a note.

Usage:

  • This Logical Model Profile is not used by any profiles in this Implementation Guide
