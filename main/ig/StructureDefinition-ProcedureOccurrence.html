Logical Model: Procedure Occurrence OMOP Table

Official URL: https://interop.esante.gouv.fr/ig/fhir/eds/StructureDefinition/ProcedureOccurrence Version: 0.1.0
Draft as of 2024-07-12 Computable Name: ProcedureOccurrence

This table contains records of activities or processes ordered by, or carried out by, a healthcare provider on the patient with a diagnostic or therapeutic purpose.

Usage:

  • This Logical Model Profile is not used by any profiles in this Implementation Guide
